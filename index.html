동유럽 여행 2026

두 폰에서 실시간으로 여행 정보 공유

1 Firebase 무료 DB 만들기

console.firebase.google.com → 구글 로그인
"프로젝트 추가" → 이름 입력 → 계속
왼쪽 "빌드" → "Realtime Database" → "데이터베이스 만들기"
"테스트 모드" 선택 → 상단 URL 복사

2 Gmail 자동 연동 설정 (Google Client ID)

console.cloud.google.com → 새 프로젝트 만들기
"API 및 서비스" → "라이브러리" → Gmail API 검색 → 사용
"OAuth 동의 화면" → 외부 → 앱 이름 입력 → 범위에 gmail.readonly 추가
"사용자 인증 정보" → "OAuth 2.0 클라이언트 ID" → 웹 애플리케이션
승인된 JavaScript 원본에 내 Netlify URL 추가 → 클라이언트 ID 복사

3 Anthropic API 키 (메일 파싱용)

console.anthropic.com → "API Keys" → "Create Key" → 복사

4 앱 연결 설정

Gmail 메일함에서 예약 메일을 자동으로 가져오는 데 사용
예약 메일 내용을 AI로 파싱하는 데 사용. 기기에만 저장됨.
